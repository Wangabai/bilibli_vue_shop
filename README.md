### 上传
git status 查看修改的文件
git add . 上传所有已经修改的文件
git commit -m '***'  保存到此用户的本地仓库
git branch 查看分支
git branch login 
git checkout master 选择主分支
git merge login 合并主分支和分支
git push 把代码推送到云端
git checkout login 切换到login分支
git push -u origin login 在云端创建login分支并推送
